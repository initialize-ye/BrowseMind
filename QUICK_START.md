# 🚀 BrowseMind 快速使用指南

## 📦 5分钟快速上手

### 第一步：安装插件

1. 下载项目
```bash
git clone https://github.com/initialize-ye/BrowseMind.git
```

2. 打开 Chrome 浏览器，访问 `chrome://extensions/`

3. 开启右上角的"**开发者模式**"

4. 点击"**加载已解压的扩展程序**"

5. 选择 `BrowseMind` 文件夹

6. ✅ 完成！工具栏会出现 🧠 图标

### 第二步：开始使用

1. **正常浏览网页**
   - 插件会自动记录你的浏览行为
   - 无需任何操作

2. **查看统计**
   - 点击工具栏的 🧠 图标
   - 查看今日和 7 天的浏览统计

3. **切换图表**
   - 点击"分类占比"、"时间分布"、"每日趋势"
   - 查看不同维度的数据可视化

### 第三步：启用高级功能（可选）

#### 启动后端服务

```bash
cd backend
pip install -r requirements.txt
python main.py
```

后端服务启动后，你可以使用：
- ☁️ 云端同步
- 🤖 AI 智能分析
- ⏰ 时间黑洞检测
- 📈 注意力曲线
- 🎯 目标设置和提醒

## 🎯 核心功能使用

### 1️⃣ 查看浏览统计

**今日统计：**
- 访问次数
- 浏览时长

**7天统计：**
- 总访问次数
- 独立网站数量

**分类统计：**
- 📚 学习
- 💻 编程
- 🎮 娱乐
- 💬 社交
- 🔧 工具

### 2️⃣ 数据可视化

**饼图 - 分类占比**
- 一眼看出时间分配
- 各分类时长占比

**柱状图 - 时间分布**
- 24小时活跃度
- 找出高峰时段

**折线图 - 每日趋势**
- 7天浏览趋势
- 观察习惯变化

### 3️⃣ 云端同步（需要后端）

1. 确保后端服务运行中
2. 点击"☁️ 同步到云端"
3. 首次同步会自动生成用户 ID
4. 数据保存到云端数据库

**好处：**
- 数据永久保存
- 跨设备访问
- 支持 AI 分析

### 4️⃣ AI 智能分析（需要后端 + API Key）

**配置 AI：**
```bash
cd backend
cp .env.example .env
# 编辑 .env，填入 DeepSeek API Key
```

**使用 AI：**
1. 点击"🤖 AI 分析"
2. 等待 5-10 秒
3. 查看分析报告：
   - 📝 行为总结
   - ⚠️ 发现的问题
   - 💡 优化建议

**示例报告：**
```
📝 行为总结
你在过去7天主要浏览编程和学习类网站，占总时长的65%。

⚠️ 发现的问题
• 娱乐时间过长，每天平均2.5小时
• 深夜浏览频繁，影响睡眠质量

💡 优化建议
• 设置娱乐时间限制，每天不超过1.5小时
• 晚上10点后减少屏幕时间
```

### 5️⃣ 高级分析（需要后端）

**时间黑洞检测：**
1. 点击"📊 高级分析"
2. 查看"⏰ 时间黑洞"
3. 识别浪费时间的网站

**注意力曲线：**
1. 点击"📊 高级分析"
2. 查看"📈 注意力曲线"
3. 找出高效时段

### 6️⃣ 目标设置（需要后端）

**创建目标：**
1. 点击"➕ 添加目标"
2. 选择目标类型：
   - 每日学习时长
   - 每日娱乐时长限制
   - 每日编程时长
   - 每日社交时长限制
3. 输入目标时长（分钟）
4. 保存

**追踪进度：**
- 实时显示当前进度
- 进度条可视化
- 状态标签：
  - 🟢 进行中
  - 🟡 即将超标
  - 🟢 已完成

**自动提醒：**
- 目标达成：🎉 恭喜完成目标
- 即将超标：⚠️ 注意时间控制

## ⚙️ 配置说明

### 仅使用插件（无需后端）

**可用功能：**
- ✅ 浏览数据采集
- ✅ 本地统计分析
- ✅ 数据可视化
- ✅ 分类统计

**不可用功能：**
- ❌ 云端同步
- ❌ AI 分析
- ❌ 高级分析
- ❌ 目标设置

### 使用后端（本地）

**启动后端：**
```bash
cd backend
pip install -r requirements.txt
python main.py
```

**可用功能：**
- ✅ 所有插件功能
- ✅ 云端同步
- ✅ 高级分析
- ✅ 目标设置
- ⚠️ AI 分析（需要 API Key）

### 配置 AI 分析

**1. 获取 API Key：**
- DeepSeek：https://platform.deepseek.com/
- OpenAI：https://platform.openai.com/

**2. 配置环境变量：**
```bash
cd backend
cp .env.example .env
```

编辑 `.env` 文件：
```env
AI_API_KEY=your_api_key_here
AI_PROVIDER=deepseek  # 或 openai
```

**3. 重启后端：**
```bash
python main.py
```

## 🐛 常见问题

### Q: 插件无法加载？
**A:** 确保开启了"开发者模式"，并选择正确的文件夹。

### Q: 数据不显示？
**A:** 刚安装需要浏览一些网页后才有数据，点击"刷新数据"。

### Q: 同步失败？
**A:** 检查后端服务是否启动：`curl http://localhost:8000`

### Q: AI 分析失败？
**A:** 检查 `.env` 文件是否配置了有效的 API Key。

### Q: 通知不显示？
**A:** 检查 Chrome 通知权限是否开启。
